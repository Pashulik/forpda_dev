Установка стилей
1. Автоматически.
  1. Плагин для быстрой установки
    Примечание:
    Требуется изменить путь папки данных приложения.
    Боковое меню клиента: Настройки программы > Системное > Путь к папке с данными
    > "Внешняя память" > Ok

2. Вручную.
  1. Входите в боковое меню клиента: Настройки программы > Системное > Путь к папке с данными
  2. Нажимаете "Внешняя память" - изменение пути к папке с данными приложения
  3. Переходиите на поле "Ручной ввод" (строка ниже) запоминаете/копируете путь
  4. Жмете OK
  5. Открываете любой файловый менеджер, проходите по пути который вы узнали через манипуляции выше, часть его должна быть - .../Android/data/org.softeg.slartus.forpdaplus/files/
    для беты - .../Android/data/org.softeg.slartus.forpdaplus.beta/files/
  6. Создайте папку styles/ и в ней light/, dark/ и black/ ВАЖНО, что бы названия папок были в нижнем регистре (без заглавных букв) иначе стили могут не отобразиться в списке клиента.
  7. Затем скачиваете стили с форума, либо берёте заранее сохранённые где-либо и копируете:
    Светлые - в папку light/
    Тёмные - в папку dark/
    Черные - в папку black/

Справочник по CSS

Справочник по каскадным таблицам стилей от сайта htmlbook.ru.
Описание программы
Описание каждого стилевого свойства происходит по одному шаблону, содержит таблицу браузеров с номерами версий, которые поддерживают указанное свойство, синтаксис написания, допустимые значения, наследование и к каким элементам свойство применяется. Каждое свойство иллюстрируется примером, который показывает область его применения.
В большинстве случаев приводится рисунок, демонстрирующий результат примера, что позволяет наглядно понять, как действует то или иное свойство CSS.
